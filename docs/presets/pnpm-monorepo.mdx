# Preset: pnpm-monorepo

Best for pnpm workspace monorepos.

## Generated workflows

- `ci.yml`
- `scheduled.yml`
- `commitlint.yml`

## Typical command

```bash
npx solvely-launchpad init --preset pnpm-monorepo --yes
```

## Notes

- If install auto-detection is not correct for your workspace, set `with.install-command` explicitly.
- Useful first override example: `pnpm install --frozen-lockfile`.
